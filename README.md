# 🐍 Pro-Classic Snake Game
A professional implementation of the classic Snake game using Python and the Turtle library. This project demonstrates core programming concepts such as real-time event loops, coordinate-based collision detection, and dynamic game states.

# ✨ Features
Dynamic Speed Increase: The game gets progressively harder as you score, increasing the challenge.

Infinite Edge Logic: The snake wraps around the screen, allowing for continuous gameplay without border collisions.

Session Management: Built-in Pause (P) and Restart (R) functions for a smoother player experience.

High-Score Tracking: Real-time scoreboard that tracks your personal best in the current session.

Visual Feedback: Color-changing scoreboard and game-over messages.

# 🛠️ Technical Skills Demonstrated
Object-Oriented Logic: Managing snake segments as a list of independent objects.

Event Handling: Using keyboard bindings to control game direction and state.

Game Mathematics: Using distance-based collision detection and coordinate randomization.
